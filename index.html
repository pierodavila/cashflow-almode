Como usar

1

Configure a API

Insira seu domínio Almode, token Bearer do admin e CNPJ da loja no painel lateral.

2

Sincronize os clientes

Vai em Sincronizar e clique em Sincronizar Clientes. O sistema puxa todos via /clientes/resumido-por-loja.

3

Caixa usa o Pop-up

Clique em Abrir Pop-up Caixa. A janela fica sempre na frente. Vendedor digita o CPF e vê o saldo na hora.

Arquitetura do sistema

→

almode.js

Único ponto de contato com a API. O agente vai preencher os stubs de cashback nativo aqui.

→

store.js

Estado global reativo. Persiste config e clientes no localStorage.

→

app.js

Controller que conecta UI ↔ store ↔ Almode. Toda lógica fica aqui, separada do HTML.

Como funciona

→

Consulta em tempo real

Primeiro busca no cache local (instantâneo). Se não encontrar, consulta a API Almode direto — campo disponivelEmCashback.

→

Arquivo standalone

O popup-vendedor.html abre direto no navegador do caixa. No app Electron, abre como janela separada que fica sempre na frente.

→

Atalho F2

No app desktop, pressione F2 em qualquer tela para abrir o pop-up do vendedor rapidamente.

Clientes

Cashback nativo — a mapear pelo agente
