📌 Passo 1 — Instalar o Botão de Sincronização (só uma vez)

1

Arraste o botão abaixo para a barra de favoritos

Se a barra não aparecer, pressione Ctrl+Shift+B (Windows) ou Cmd+Shift+B (Mac)

📥 Sync Aeropostale → CashFlow
⬆️ Arraste este botão para a sua barra de favoritos do Chrome
Depois é só clicar nele quando estiver no Millennium com o relatório aberto
💡 Como instalar: Arraste o botão verde acima para a barra de favoritos do Chrome (a barra de links no topo do browser)

🚀 Passo 2 — Usar no Dia a Dia

1

Abra o Millennium

Acesse app.aeropostale.com.br:6017 normalmente

2

Gere o relatório CLIENTES_VENDAS_CELULAR_EMAIL

Selecione o período desejado (recomendado: últimos 30 dias para atualizações rápidas) e clique em Visualizar

3

Clique no favorito "📥 Sync Aeropostale → CashFlow"

O script vai extrair os dados do PDF automaticamente e enviar pro Supabase. Uma janela vai confirmar quando terminar.
